About Project

This project focuses on early detection of Alzheimer’s disease using deep learning techniques. It analyzes MRI brain images to identify patterns associated with cognitive decline.

The system uses advanced models like CNN, MobileNet, and VGG16 to improve classification accuracy.

  • 🧠 Mild Demented
  • 🧠 Moderate Demented
  • 🧠 Non Demented
  • 🧠 Very Mild Demented

Developed using TensorFlow, Keras, and Flask for real-time prediction.
